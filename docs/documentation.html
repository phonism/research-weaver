📚 Documentation

Complete guide to using Research Weaver, an AI-powered multi-agent research system.

🚀 Installation

Prerequisites

  • Python 3.8 or higher
  • DeepSeek API Key (get from platform.deepseek.com)
  • Tavily API Key (get from tavily.com)

Install from Source

# Clone the repository
git clone https://github.com/phonism/research-weaver.git
cd research-weaver

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env with your API keys

Environment Variables

Create a .env file in the project root with the following variables:

# Required API Keys
DEEPSEEK_API_KEY=your_deepseek_api_key_here
TAVILY_API_KEY=your_tavily_api_key_here

# Optional
SERPER_API_KEY=your_serper_api_key_here  # Fallback search engine
💡 Tip: You can use either Tavily or Serper for web search. Tavily is recommended for better results.

⚡ Quick Start

1. Run the Web Interface

streamlit run ui/app.py

Open your browser to http://localhost:8501 and start researching!

2. Programmatic Usage

import asyncio
from src.supervisor import SupervisorAgent
from src.tools import create_llm_client, create_tools
from src.memory_store import MemoryStore

async def main():
    # Create components
    llm_client = create_llm_client()
    tools = create_tools()
    memory_store = MemoryStore()
    
    # Create supervisor
    supervisor = SupervisorAgent(
        llm_client=llm_client,
        model="deepseek-chat",
        tools=tools,
        memory_store=memory_store
    )
    
    # Conduct research
    result = await supervisor.research(
        "Analyze the latest trends in renewable energy technology"
    )
    
    print(result)

# Run the research
asyncio.run(main())

⚙️ Configuration

Model Configuration

Research Weaver uses DeepSeek by default, but you can configure different models:

supervisor = SupervisorAgent(
    llm_client=llm_client,
    model="deepseek-chat",  # or "deepseek-coder"
    tools=tools,
    memory_store=memory_store
)

Search Configuration

Configure search tools with custom settings:

from src.tools import SearchTool, ReadTool

# Custom search tool
search_tool = SearchTool(api_key="your_tavily_key")

# Create tools dictionary
tools = {
    "search": search_tool,
    "read": ReadTool()
}

📖 Usage

Research Process

Research Weaver follows a structured multi-agent approach:

  1. Query Analysis - Supervisor analyzes your research question
  2. Agent Creation - Specialized researchers are created for different aspects
  3. Information Gathering - Agents search and read relevant sources
  4. Synthesis - Results are combined into a comprehensive report

Research Examples

✅ Good Research Queries:
  • "Analyze investment opportunities in renewable energy sector"
  • "Current trends in artificial intelligence and machine learning"
  • "Impact of climate change on agricultural production"
⚠️ Less Effective Queries:
  • "What is AI?" (too basic)
  • "Everything about technology" (too broad)
  • "Current weather" (real-time data)

🔧 API Reference

SupervisorAgent

Main orchestrator of the research process.

class SupervisorAgent:
    def __init__(self, llm_client, model, tools, memory_store):
        """Initialize supervisor with required components."""
        
    async def research(self, query: str) -> str:
        """Conduct research on a topic and return final report."""
        
    def get_research_summary(self) -> dict:
        """Get summary of research activities."""

ResearcherAgent

Specialized agent for focused research tasks.

class ResearcherAgent:
    def __init__(self, role, initial_task, llm_client, model):
        """Initialize researcher with specific role and task."""
        
    async def research_task(self, task: str) -> str:
        """Execute specific research task."""

💡 Examples

Basic Research

import asyncio
from src.supervisor import SupervisorAgent
from src.tools import create_llm_client, create_tools
from src.memory_store import MemoryStore

async def basic_research():
    # Setup
    llm_client = create_llm_client()
    tools = create_tools()
    memory_store = MemoryStore()
    
    supervisor = SupervisorAgent(
        llm_client=llm_client,
        model="deepseek-chat",
        tools=tools,
        memory_store=memory_store
    )
    
    # Research
    result = await supervisor.research(
        "What are the key trends in electric vehicle adoption?"
    )
    
    return result

# Run research
result = asyncio.run(basic_research())
print(result)

Custom Research with Memory

async def advanced_research():
    # Setup with persistent memory
    memory_store = MemoryStore()
    
    # Conduct multiple related research sessions
    queries = [
        "Current state of electric vehicle technology",
        "Electric vehicle market trends and forecasts",
        "Challenges facing electric vehicle adoption"
    ]
    
    results = []
    for query in queries:
        supervisor = SupervisorAgent(
            llm_client=create_llm_client(),
            model="deepseek-chat",
            tools=create_tools(),
            memory_store=memory_store  # Shared memory
        )
        
        result = await supervisor.research(query)
        results.append(result)
    
    return results

🔧 Troubleshooting

Common Issues

API Key Errors

ValueError: Please set DEEPSEEK_API_KEY environment variable

Solution: Make sure your .env file contains valid API keys and is in the project root directory.

Search Tool Issues

Error: Please set TAVILY_API_KEY or SERPER_API_KEY environment variable

Solution: Add at least one search API key to your .env file.

Memory/Performance Issues

If research takes too long or uses too much memory:

  • Limit research steps: await supervisor.research(query, max_steps=10)
  • Use more specific queries to reduce search scope
  • Clear cache: rm -rf ~/.research_weaver_cache

Debug Mode

Enable debug logging to see detailed research process:

import logging
logging.basicConfig(
    level=logging.DEBUG,
    format='%(asctime)s - %(levelname)s - %(message)s'
)
